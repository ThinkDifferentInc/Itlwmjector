# Itlwmjector

> **Easily inject your Wi-Fi profile into itlwm.**

`Itlwmjector` is a quick and easy to use tool to insert your Wi-Fi profiles into itlwm.

## 🚀 Getting Started

### Prerequisites

You need Python 3 installed on your system, along with the `rich` library for the terminal UI.

```bash
pip install rich

```

### Usage

1. Clone or download this repository.
2. Run the script from your terminal:

```bash
python Itlwmjector.py

```

3. Follow along.
4. The script will append your Wi-Fi profile into the Info.plist file of the kext.
